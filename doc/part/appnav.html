固定应用导航条

应用导航条通常需要固定在页面顶部或底部。在 <body> 或 .page 上使用辅助类 .with-appbar-top 或 .with-appbar-bottom，并在 .appnav 上使用辅助类 .affix.dock-top 或 .affix.dock-bottom。

// 固定在顶部
<body class="with-appbar-top">
  <header class="appbar heading affix dock-top">
    ...
  </header>
</body>
// 固定在底部
<body class="with-appbar-bottom">
  <footer class="appbar heading affix dock-bottom">
    ...
  </footer>
</body>

在同一个页面可以同时将两个应用导航条分别固定顶部和底部。

// 固定在顶部和底部
<body class="with-appbar-top with-appbar-bottom">
  <header class="appbar heading affix dock-top">
    ...
  </header>
  ...
  <footer class="appbar heading affix dock-bottom">
    ...
  </footer>
</body>

当应用导航栏固定时，可以为 .appnav 使用 .dock-auto 来让固定的应用导航栏响应页面滚动事件并在合适的时候自动隐藏和显示。

// 固定在顶部和底部并自动显示或隐藏
<body class="with-appbar-top with-appbar-bottom">
  <header class="appbar heading affix dock-top dock-auto">
    ...
  </header>
  ...
  <footer class="appbar heading affix dock-bottom dock-auto">
    ...
  </footer>
</body>
